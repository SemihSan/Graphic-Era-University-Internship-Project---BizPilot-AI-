# Source Code

Future code will be organized around the MVP modules:

- RAG ingestion and retrieval
- Lead scoring:
  - `lead_scoring_baseline.py`: Kaggle dataset'i temizler, Logistic Regression baseline modeli egitir ve rapor olusturur.
  - `lead_scoring_predictor.py`: Egitilmis modeli yukler, yeni lead icin 0-100 arasi skor uretir.
- Agentic routing and outreach
- Competitor intelligence retrieval
- Streamlit UI
- RAGAS evaluation
